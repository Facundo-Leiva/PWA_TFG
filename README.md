
Este proyecto surge como trabajo de tesis y consiste en el desarrollo de una herramienta tecnológica orientada a la participación ciudadana, cuyo objetivo es facilitar el reporte de incidencias urbanas de diversa índole (infraestructura, servicios, seguridad, entre otros) para su posterior gestión y resolución por parte de las autoridades competentes.

La plataforma permite que cualquier ciudadano se registre y cree su propio perfil, desde el cual puede visualizar reportes generados por otros usuarios, interactuar con ellos mediante comentarios u otras acciones, y crear nuevos reportes de forma sencilla e intuitiva. Cada incidencia puede ser georreferenciada, lo que posibilita el uso de mapas interactivos para visualizar la distribución de los reportes en el territorio y facilitar su análisis.

El sistema promueve la colaboración y la transparencia, fomentando una comunicación más directa entre la ciudadanía y las instituciones responsables, y contribuyendo a una mejor identificación y seguimiento de las problemáticas urbanas.
